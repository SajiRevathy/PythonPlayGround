NUMBER GUESSING GAME
    It is a simple number guessing game where computer randomly generate a number upto 1 to 100, then ask the user to guess the number.

    If the guess is above the number,
    then computer will say "Sorry Too High"
    If the guess is below the number,
    then computer will say "Sorry Too Low"
    If the guess is right,
    then computer will say "Correct! You  got it in ____ attempts, well done."

    System will track the attempts of users, so it will mention on which attempt users guessed the correct answer.

    In Main.py
    Players are restricted with limited attempts. so they will get only 4 chances to guess.

    In GameVersion.py
    Players get infinite chances. They can play until they win.
